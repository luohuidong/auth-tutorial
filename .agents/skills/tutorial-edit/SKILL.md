## 编写新的教程内容需要遵循的步骤和规范

1. 获取教程目录结构，确保新教程内容能够正确地放置在目录结构中，并且符合目录结构的规划。
2. 参照 [认证教程规则](./references/tutorial-rules.md) 确保在编写教程相关内容时遵循这些规则。
3. 对教程内容进行规划
   1. 确定教程主体内容和结构，确保内容清晰、易于理解，并且符合目标受众的需求。
   2. 如果教程中涉及大量示例代码，应规划将代码编写到对应章节的 code 目录，而不是将示例代码全部放在教程 md 文件中，以保持教程内容的简洁和可读性。
4. 编写教程内容
5. 参照目录大纲中的已完成的教程数据，更新 vitepress 配置文件中的侧边栏配置，以确保已完成的教程内容能够正确显示在侧边栏中。
6. 如果需要将示例代码编写到对应章节的 code 目录，查询一下教程示例代码所使用的技术栈，确保示例代码的技术栈与教程内容相匹配。

## 如果是编辑已有教程内容，需要遵循的规范

markdown 内容风格参照 [Markdown Style Rules](./references/markdown-style-rules.md)

## 编新写教程跟编辑已有教程都需要遵守的规则

- 内容不能胡编乱造，必要的时候查询相关资料，确保内容的准确性和可靠性。
