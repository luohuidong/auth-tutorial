---
name: tutorial-create
description: 编写新的教程内容时需要执行的步骤
---

1. 加载 [认证教程规则](../../../rules/tutorial-rules.md) 确保在编写教程相关内容时遵循这些规则。
2. 对教程内容进行规划
   1. 确定教程主体内容和结构，确保内容清晰、易于理解，并且符合目标受众的需求。
   2. 如果教程中涉及大量示例代码，应规划将代码编写到对应章节的 code 目录，而不是将示例代码全部放在教程 md 文件中，以保持教程内容的简洁和可读性。
3. 编写教程内容
4. 参考 [教程目录结构规范](../../../rules/structure-rules.md)，更新教程大纲。
5. 参照目录大纲中的数据，更新 vitepress 配置文件中的侧边栏配置，以确保已完成的教程内容能够正确显示在侧边栏中。
6. 如果需要将示例代码编写到对应章节的 code 目录，应遵循 [代码示例规范](../../../rules/code-example-rules.md)。
