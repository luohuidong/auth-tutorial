# Tutorial Check Skill

## 第一步：检查教程文章的质量

- 检查教程整体逻辑是否清晰，是否有明显的逻辑漏洞。
- 检查教程的结构安排是否合理，是否有清晰的段落划分和标题层次。
- 检查教程的内容是否准确，是否有事实错误或者误导性的表述。
- 检查教程中所举例子是否正确，是否能够有效地说明问题。
- 教程内容有不确定的地方，不能胡编乱造，应该进行网络搜索查证，确保信息的准确性。

## 第二步：检查有没有乱用 markdown 语法的情况

如使用 `**` 来做小节标题，而不是用来强调文字。

## 第三步：如果文章存在问题，则指出问题所在，并进行修改

1. 当教程文章存在质量问题应该指出具体的问题所在，并提供改进建议，然后与用户进行沟通，确保用户理解问题所在以及改进的方向。
2. 根据与用户沟通的结果，生成教程优化的具体计划，再与用户进行确认，确保优化计划的可行性和有效性。
3. 根据确认的优化计划，对教程文章进行修改和优化
