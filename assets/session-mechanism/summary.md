本篇系统讲解 Session 认证机制的所有核心组件。内容涵盖：Session 与 Cookie 的关系（Session ID 存于 Cookie，Session 数据存于 Redis）；Cookie 五大安全属性（HttpOnly 防 XSS 劫持、Secure 防中间人攻击、SameSite 防 CSRF、Domain/Path 的作用域控制、Max-Age 过期策略）；Redis Session 存储的必要性及 `connect-redis` + `express-session` 的配置要点；Passport.js Local 策略在 NestJS 中的完整集成（LocalStrategy、SessionSerializer、LocalAuthGuard、SessionAuthGuard）；`serializeUser`/`deserializeUser` 的调用时机与作用；以及时序攻击防御、Session Fixation 防御、注销正确实现等安全加固措施。
