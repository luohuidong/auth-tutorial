# 教程内容组织

- 对应章节的教程，存放在根目录 `docs` 文件夹下。例如 `foo` 章节教程存放在 `docs/foo.md` 中。
- 其他对应章节的相关内容，例如章节教程内容要求、章节对应的可执行代码等，存放在 `assets` 文件夹下的对应章节文件夹中。例如 `foo` 章节的相关内容存放在 `assets/foo/` 文件夹下。

```textplain
docs/
├── foo.md # 章节内容
tutorials/
├── foo/ # 章节文件夹，名称与章节名称一致
|   ├── summary.md # 章节内容摘要
│   ├── source # 章节对应的可运行代码
│   └── ...    # 其他内容
├── ...
```
